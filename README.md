# Automatic Control Systems Projects

## Project Description

This repository contains a collection of projects developed as part of an Automatic Control course. The projects focus on modeling, analysis, and control design of different dynamic systems using classical and modern control techniques.

The aim is to understand system behavior, design appropriate controllers, and evaluate performance through simulation.

## Technologies Used

* MATLAB
* Simulink
* Control System Toolbox
  
## Projects Overview

### DC Motor System Modeling and Analysis

* Mathematical modeling of a DC motor
* Transfer function derivation
* Time response analysis (step response, stability)

### P, PI, PD Controller Design for DC Motor

* Design of classical controllers (P, PI, PD)
* Performance comparison (rise time, overshoot, steady-state error)
* Controller tuning and response evaluation

### State-Space Modeling, Analysis and Control

* State-space representation of dynamic systems
* Controllability and observability analysis
* State feedback controller design

#### Subsystems:

* **Suspension System**

  * Modeling of a mass-spring-damper system
  * Ride comfort and stability analysis

* **Ball and Beam System**

  * Nonlinear system modeling
  * Linearization and control design
  * Position stabilization

### Flexible Satellite Attitude Control

* Modeling of flexible satellite dynamics
* Attitude stabilization problem
* Control design considering system flexibility

## How to Run

1. Open MATLAB
2. Navigate to the desired project folder
3. Run the corresponding open ".slx" model
4. Execute simulations and analyze results
